What is QR code ?

A QR (quick response) code is a type of 2D barcode, made up of a 2-tone pixel pattern. Unlike a traditional barcode, a QR code is able to contain more information and is scannable from any orientation. They were first used by Denso Wave, who developed them for marketing components, but soon found their way into mobile marketing when smartphones were more widely adopted.
